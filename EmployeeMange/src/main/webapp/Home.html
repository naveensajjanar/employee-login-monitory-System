Developer


  A developer is an individual that builds and create software
nd applications. He or she writes, debugs and executes the source code of a software application.
A developer is also known as a software developer, computer programmer, programmer,
software coder or software engineer........

  Developers perform important roles in creating, testing, building and optimizing
applications, software and computer systems. There are many developer types. Learning the differences
between these types can help you discover the type of development you're most interested in pursuing.
In this article, we explain what a developer is,list some of the most common developer types
and offer tips for becoming a developer.
